# Project Overview

This is a simple Node.js web server project.

## Project Structure

- `server.js` - Simple HTTP server serving static files on port 5000
- `index.html` - Welcome page

## Running the Project

The web server runs with `node server.js` and listens on `0.0.0.0:5000`.
